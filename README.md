## 📋 Project Overview

Iron-Haven-Gym offers a simple way for members to track their strength gains and fitness progress. You use this software to log your daily workouts, monitor personal records, and view trends in your lifting capability. This tool runs directly on your Windows desktop. It stores your information locally on your computer to keep your fitness data private.

* Memory: 4 gigabytes of RAM

## 🚀 Using the Software

Iron-Haven-Gym features a clean interface built for quick data entry. 

**Logging a Workout**
Navigate to the Add Workout tab. Select the type of exercise you performed, such as bench press, squat, or deadlift. Enter the weight in pounds or kilograms and the number of repetitions completed. Save your entry to record the session. 

**Viewing Progress**
Select the Analytics tab to view your history. The software generates graphs based on your saved entries. You can filter these views by date range or specific exercise types. This helps you notice patterns in your strength over several weeks or months.

**Managing Settings**
The Settings menu allows you to adjust your personal preferences. You can switch between imperial and metric units. You can also import or export your historical data if you need to move your records to a new computer.

## 🛠 Troubleshooting Common Issues

If you encounter problems, review these solutions:

* Software fails to launch: Ensure you have administrative rights on your Windows machine. Right-click the icon and choose Run as Administrator.
* Missing data: Verify that the application has permission to write files to your Documents folder. Check that your antivirus program does not block the application from saving your local data files.
* Graphics issues: Ensure your Windows display scaling is set to 100 percent. Some high-resolution monitors cause visual distortions if scaling exceeds this point.
* Updates: Periodically check the main project page for version updates. Uninstall the existing version before installing a newer package to prevent file conflicts.

## 🛡 Data Security

All your information remains on your local hard drive. The application does not send your workout history to external servers or cloud providers. You maintain full control over your files. Back up your data folder regularly by copying it to an external drive or a secure cloud service of your choice.
